# Proscan

Application security platform. Twelve scanners, one deployment, your infrastructure.

SAST · DAST · SCA · Secrets · IaC · Container · API · AI/LLM · Binary · Network · Code Quality · ProScan Deep

---

### What it does

Proscan finds vulnerabilities across your entire software stack — source code, running applications, dependencies, containers, APIs, infrastructure templates, and AI endpoints. Everything runs on-premises. Nothing leaves your network.
